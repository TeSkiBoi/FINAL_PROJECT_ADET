# User Guide - Inventory Management System

## Getting Started

### Logging In
1. Launch the application
2. Enter your credentials:
   - Default username: admin
   - Default password: admin123
3. Use the "Show Password" checkbox to verify your password
4. Click "Login" or press Enter

### Dashboard Overview
The dashboard provides a quick overview of your inventory system:

1. **Navigation Menu (Left Sidebar)**
   - Home: Return to main dashboard
   - Products: Manage inventory items
   - Suppliers: Manage supplier information
   - Transactions: View transaction history
   - Reports: Generate system reports
   - Logout: Safely exit the system

2. **Statistics Cards**
   - Total Products: Current inventory count
   - Total Suppliers: Number of registered suppliers
   - Category Breakdown: Products by category

### Managing Products

1. **Adding a New Product**
   - Click "Products" in the sidebar
   - Fill in the required fields:
     * Product Name
     * Category (select from dropdown)
     * Quantity
     * Price
   - Click "Add Product"

2. **Searching Products**
   - Use the search bar at the top
   - Enter product name or category
   - Results update automatically

3. **Updating Products**
   - Select product from the table
   - Modify details in the form
   - Click "Update"

4. **Deleting Products**
   - Select product from the table
   - Click "Delete"
   - Confirm deletion

### Managing Suppliers

1. **Adding a New Supplier**
   - Click "Suppliers" in the sidebar
   - Enter supplier details:
     * Name
     * Contact
     * Address
     * Status
   - Click "Add Supplier"

2. **Searching Suppliers**
   - Use the search bar
   - Search by name, contact, or address
   - Results show in real-time

3. **Updating Supplier Information**
   - Select supplier from the table
   - Modify information
   - Click "Update"

4. **Managing Supplier Status**
   - Select supplier
   - Change status (Active/Inactive)
   - Update to save changes

### System Security

1. **Password Management**
   - Use strong passwords
   - Change password regularly
   - Keep credentials secure

2. **Safe Logout**
   - Always use the Logout button
   - Confirm logout when prompted
   - Wait for confirmation message

### Troubleshooting

1. **Common Issues**
   - Login problems: Check caps lock
   - Search not working: Clear and retry
   - Table not updating: Refresh panel

2. **Error Messages**
   - Read error description
   - Follow suggested solutions
   - Contact admin if persistent

2. **System Usage**

3. **Security**
